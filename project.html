두 개의 마스터를 이용한 Teaching형 탐사로봇

프로젝트 진행시기 : 2018년 3월 ~ 2018년 6월
프로젝트 진행기간 : 4개월
프로젝트 구성 인원 수 : 2명
결과 : 캡스톤 디자인 경진대회 장려상
내가 맡은 역할 : software master(3D) 제작
로봇의 용도 : 사람이 들어가기 어려운곳을 들어가서 작업을 수행하는 로봇
특이점 : 비전문가도 쉽게 활용할 수 있게 직관적으로 그래픽을 조종하거나 똑같은 형태의 하드웨어를 조종하는 형태의 조작방식을 구현했다. 활용 기술 : Unity, Processing, Arduino

인디언즈

프로젝트 진행시기 : 2018년 03월 ~ 2019년 06월 프로젝트 진행기간 : 1년 이상
프로젝트 구성 인원 수 : 기획 2명, 영업 2명
결과 :
- 기술혁신형 창업기업 지원사업 유치 (5800만원)
- 세종창조경제혁신센터 아이디어무브 크라우드 펀딩 최우수상 수상
내가 맡은 역할 : CEO - 경영, 웹 기획, PR, 시장분석 등
프로젝트 설명 : 인디게임과 인디크리에이터를 잇는 퍼블리싱 플랫폼
www.indiens.io
임시 서비스 중단
특이점 :
3차례의 피봇
기존 아이템 : 인디게임들을 하나의 메이커로 묶어주는 인디게임 소속사
1차 피봇 : 인디게임과 마케터를 잇는 퍼블리싱 플랫폼
2차 피봇 : 인디게임과 인디크리에이터를 잇는 퍼블리싱 플랫폼
3차 피봇 : 인디들의 게임제작 플랫폼

가제트 만능팔

프로젝트 진행시기 : 2017년 12월 ~ 2018년 02월 프로젝트 진행기간 : 3개월
프로젝트 구성 인원 수 : 기획 1명, 개발 3명
결과 :
- KU동계 집현전 캡스톤디자인 공모전 최우수상
내가 맡은 역할 : Arduino와 Processing을 활용한 2d 그래픽 마스터 제작
프로젝트 설명 : master - slave동작을 하는 로봇팔 제작
특이점 : 이 프로젝트를 프로세싱을 통한 2d로 제작한 이후
"Teaching형 탐사로봇" 프로젝트에서 Unity를 이용하여 3d화로 발전.
